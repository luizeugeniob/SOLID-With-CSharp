# E-Auction, seu melhor Leilão Online!

E-Auction é um projeto em [ASP.NET Core](https://docs.microsoft.com/pt-br/aspnet/core) utilizado como base para realizar o curso [SOLID com C#](https://cursos.alura.com.br/course/solid-csharp-principios-orientacao-a-objetos) da Alura. 

Neste curso, é realizada uma carga inicial com um projeto que não implementa o conceito SOLID, e ao longo do curso são realizadas refatorações a fim de adequar o projeto ao padrão SOLID.
